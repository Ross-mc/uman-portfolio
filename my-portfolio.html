My name is Ross McIntyre and I am an aspiring web developer and software engineer.

I am currently enrolled on the University of Manchester Full Stack Coding Bootcamp and I have also been self-learning through various resources such as Free Code Camp and Udemy.

I have experience with front end technologies such as HTML/CSS, CSS frameworks (such as Bootstrap) and JavaScript (with jQuery). I also have experience with back end development with Node.js (using Express.js as a framework) and MongoDB. I am also currently learning React.js and have experience with Python.

Below is a showcase of some of the projects I have been working on.

Workout Weekly

Workout Weekly is a full stack Node.js application with a MySQL database. The application is a workout and daily planner which gives the user a calendar so they can fill out their weekly work schedule and fit it workout videos. The web app utilises the Google API and a database of over 200 curated videos from some of the most popular fitness channels. The application includes a user registration and log in system.
